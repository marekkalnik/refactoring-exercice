Salut,

Voici l'exercice de refactoring dont nous avons parlé.

Objectif :
 - refactoriser le code pour appliquer les bonnes partiques de programmation
    - dans le message du commit justifier le changement
 - implementer le strike (faire passer le test déjà écrit)

Contraintes à respecter :
 - chaque commit doit modifier 3 lignes maximum
 - les tests doivent passer pour chaque commit
 - les tests existants ne peuvent pas être modifiés

Livrable :
 - repository Git avec des commits (sur GitHub ou ton local en ZIP).

Pour installer le projet execute `npm install` dans le dossier racine.
Pour lancer les test `npm run test`.

Le test avec la fonction xit est désactivé. Pour l'activer il faut enlever le "x".
